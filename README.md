# 💻 Projeto Programaria Front-End

Este projeto foi desenvolvido durante o curso **Programaria Front-End**, com o objetivo de praticar HTML, CSS e JavaScript na construção de páginas web interativas.

## ✨ Sobre o projeto

O site é composto por duas partes principais:

🎬 **Cinema de Memórias**
Uma experiência interativa com filmes e elementos visuais inspirados no universo Disney/Pixar.

👩‍💻 **Mulheres na Tecnologia**
Uma página dedicada a contar histórias e destacar a importância de mulheres que marcaram a tecnologia, como Ada Lovelace.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 🎯 Objetivo

Praticar conceitos fundamentais de desenvolvimento front-end, como:

* Estruturação de páginas com HTML
* Estilização com CSS
* Interatividade com JavaScript
